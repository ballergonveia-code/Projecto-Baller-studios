Pixel HellRun is a challenging 2D HTML5 platformer game built with JavaScript and Canvas. Players must collect coins, avoid deadly traps, defeat enemies, and survive through increasingly difficult levels. The game features pixel-art graphics, dynamic difficulty progression, multiple hazards, a timer system, and 100 levels of hardcore gameplay.

Features:

* 100 progressive levels
* Pixel-art retro style
* Coins and score system
* Enemies and traps
* Laser obstacles
* Falling blocks
* Life and timer mechanics
* Mobile and desktop controls
* Built with pure HTML, CSS, and JavaScript

Goal:
Collect all coins, survive dangerous obstacles, and reach Level 100 to win.
